  • 翼城中学

    翼城中学创建于1952年，位于县城南端。东枕浍水，南襟翔山，是宋代名相文彦博的“潞公旧治”和历代县衙故地，木牌楼、石牌坊翘首顾盼，文峰塔、五奎塔遥遥相望，明朝钟楼修葺一新坐落其中。校园古朴幽静，人文历史氛围浓郁，办学条件得天独厚。
